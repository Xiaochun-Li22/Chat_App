#  Chat App

A simple real-time chat application built with **React**, **Node.js**, **Express**, and **Socket.io**.  
Fully deployed using **Render**.

---

##  Live Demo

**Frontend:**  
https://chat-app-client-wfhj.onrender.com

**Backend:**  
https://chat-app-service-w87e.onrender.com

---

##  Features

- Real-time messaging  
- Join any chat room  
- See active users in each room  
- Mobile-friendly and responsive  
- Clean and simple UI

---

##  Tech Stack

- React (Create React App)  
- Node.js  
- Express  
- Socket.io  
- Render (deployment)

---

##  How to Use

1. Open the live frontend URL.  
2. Enter a display name.  
3. Enter a room name (must match with your friend).  
4. Share the link + room name with your friend.  
5. Start chatting in real time
